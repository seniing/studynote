#### 3.  Breakpoint

| Breakpoint        | Class infix | Dimensins  |
| ----------------- | ----------- | ---------- |
| X-Small           | `None`      | < 576 px   |
| Small             | `sm`        | >= 576 px  |
| Medium            | `md`        | >= 768 px  |
| Large             | `lg`        | >= 992 px  |
| Extra latge       | `xl`        | >= 1200 px |
| Extra extra large | `xxl`       | >= 1400 px |
